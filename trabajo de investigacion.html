Trabajo de investigación

¿Qué diferencia hay entre sitio y página web?

La página web hace referencia especificamente a documentos HTML.

Un sitio web es una colección de páginas web vinculadas (más sus recursos asociados) que comparten un único nombre de dominio.

¿Qué pasos debo seguir para crear un sitio web?

  • Planifica la idea de tu sitio web.
  • Elige la plataforma ideal para crear tu sitio web.
  • Construye la estructura de tu sitio web.
  • Diseña y crea el contenido de tu sitio web.
  • Agrega herramientas profesionales.

¿Qué es un dominio?

Un dominio de internet es un nombre único que identifica a una subárea de internet.
Las 4 mejores opciones para registrar un dominio son:

Bluehost, Domain.com, Namecheap y NameSilo.
